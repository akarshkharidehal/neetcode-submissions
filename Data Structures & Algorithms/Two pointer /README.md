The two-pointer technique uses two index variables to traverse a data structure simultaneously.
It reduces time complexity from O(n^2) to O(n) by avoiding nested loops. 
It is primarily applied through three patterns: 

a)converging pointers moving inward from opposite ends, 
b)parallel pointers moving in the same direction.

Some coding problems where this concept can be applied are 

a) Two sum
b) Container with max water
